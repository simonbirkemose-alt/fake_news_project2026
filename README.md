# fake_news_project2026
We have imported the files we are using (fake news corpus, Liar dataset etc) to a folder called Group_project if you create such a folder parallel to this folder it should be possible to execute the code, that is you place the files so they can be found using the filepath "../Group_project/relevant_file.csv". In addition to this some of the notebooks have been excecuted in google colab, meaning that to excecute these using the files, you should either redirect the path to a folder containing the relevant files, or use google drive and place the files in a path as shown in the code. Other than this we are giving a requirements.txt file so you can get the correct modules for the code. NOTE when we made our github public our hf token was made invalid, so to run the part of the code with a preexisting model you must fetch a token to run the llama model. You can find the correct token at https://huggingface.co/meta-llama/Meta-Llama-3-8B
